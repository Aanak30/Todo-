# 📝 To-Do List App (React + Vite)

A simple and interactive To-Do List application built using React.

---

## 🛠️ Tech Stack
- React (Functional Components)
- Vite
- JavaScript (ES6)
- CSS

---

## 🚀 Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Edit tasks
- Prevent duplicate tasks
- Responsive UI
